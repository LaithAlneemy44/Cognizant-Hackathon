## Brand Voice Notes

Write ForgeSpec copy that is:
- **Confident, not boastful.** State what the product does. Don't claim it's the best or first.
- **Concrete, not vague.** Name the artifact, name the stage, name the outcome.
- **Technical without being impenetrable.** Judges and developers should both nod.
- **Honest about what it is.** ForgeSpec is a pipeline, not magic.

Avoid:
- "revolutionise" / "transform" / "disrupting"
- "seamless" / "frictionless" / "effortless"
- "leveraging AI" / "powered by AI" / "AI-driven"
- "cutting-edge" / "state-of-the-art" / "next-generation"
